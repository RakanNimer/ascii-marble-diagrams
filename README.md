```bash

  cd ascii-marble-diagrams
  yarn install
```

### Usage

Comes with the following scripts : 

1. Run demo with live reload.

```bash
  yarn demo 
```

2. Build.

```bash
  yarn build
```

3. Lint.

```bash
  yarn lint
```

4. Run Tests

```bash
  yarn test
```

5. Precommit test run

Before committing all staged js(on) code is linted, prettified and tested.
